# Tea Tasting App
This was created during my time as a student at Code Chrysalis. This app can recommend the kind of tea which you want to drink. Have fun!

## Setting
1. yarn install
```bash
yarn install
```
2. create DB
```bash
yarn migrate
```
3. insert data to table
```bash
yarn seed
```
4. server setup
```bash
yarn build
```
```bash
yarn start
```
3. open `http:localhost:9000/` on your browser.
